# Ooru Trees

An interactive map of Bengaluru's trees, built from the BBMP Tree Census dataset.

## Origin

This started with a [tweet that went viral](https://x.com/viksmals/status/2025871650743566541?s=20) — a visualisation of where pink trumpet trees across Bengaluru. The BBMP census dataset already had data for multiple species, so it seemed fun to try and expand to more.

## What it does

- Browse and filter trees across hundreds of species on an interactive map
- Switch between common names (Gulmohar, Mango, Rain Tree) and scientific names
